## 🚀 Publicar en GitHub Pages

1. Crea un repositorio en GitHub (ej: `intra-boutique`)
2. Sube los archivos `index.html` y `logo.jpeg` al repositorio
3. Ve a **Settings → Pages**
4. En **Source**, selecciona `main` branch y carpeta `/ (root)`
5. Guarda — en unos minutos tu sitio estará en:
   `https://tu-usuario.github.io/intra-boutique/`
